# Why I Prefer Static Sites

Static sites remove entire classes of problems.

## Benefits

- No database
- No runtime
- No nonsense

## Comparison

| Feature | Static | Dynamic |
|---------|--------|---------|
| Speed | Fast | Slower |
| Security | Simple | Complex |
| Hosting | Cheap | Expensive |

### Performance Metrics

1. Page load time
2. Time to first byte
3. Cumulative layout shift

#### Additional Considerations

Static sites are ideal for blogs, documentation, and marketing content.
